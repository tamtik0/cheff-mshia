# AI Recipe Generation Bot

A Python-based intelligent culinary assistant that leverages the **Claude LLM** to generate personalized recipes based on user preferences and dietary constraints.

## 🚀 Features
- **Personalized Generation:** Tailors recipes to specific ingredients, cuisines, or dietary needs.
- **Intelligent Prompting:** Uses advanced prompt engineering to ensure safety and accuracy.
- **Claude API Integration:** Seamlessly connects with Anthropic's Claude for high-quality natural language processing.
- **JSON Output Handling:** Processes structured data for clean, readable recipe formatting.

## 🛠️ Technical Stack
- **Language:** Python 3.x
- **LLM:** Claude (Anthropic API)
- **Data Handling:** JSON, Environment Variables
